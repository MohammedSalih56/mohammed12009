python :


  It is an easy-to-learn, open source, extensible, high-level programming language that adopts the Object-Oriented Programming (OOP) method
  Python is an interpreted and versatile language, and it is widely used in many fields, such as building independent programs using graphical
  interfaces and in web applications, and it can be used as a scripting language to control the performance of many programs such as Blender
  In general, Python can be used to create simple programs for beginners, but also to create massive projects at the same time. It is often
  recommended for beginners in the field of programming to learn this language because it is among the fastest-learning programming languages.
